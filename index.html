ConnectN

ECS 32B, Fall 2021

Created a command line game of Connect 4 using object-oriented programming and unit testing in Python. Users can specify game parameters like board size, player pieces, and player types (two AI player types). The random AI player will place pieces randomly. The simple AI player will place pieces randomly until it sees that it can win in one move or blocks the other player from winning with one move.
